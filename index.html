Ordinal Challenge

This ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 

puts "Degrees in Celsius"
celsius = gets.chomp.to_i

def farhenheit(n)
	return (n * 1.8) + 32
end

puts "#{celsius} degrees Celsius is equal to #{farhenheit(celsius)} degrees Farhenheit"}

def ordinal(x)
  modulo = x % 10
  hundreds = x % 100
  if hundreds >= 11 && hundreds <= 13
    "th"
    elsif  modulo == 1
    "st"
    elsif modulo == 2
    "nd"
    elsif modulo == 3
    "rd"
    else
    "th"
  end
end

puts "enter a number"
answer = gets.chomp.to_i  
puts "That's the #{answer}#{ordinal(answer)} item"

Foobar

This ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 

def foobar(quantity)
  x = 0
  while x < quantity
    x = x + 1
      if x % 3 == 0 && x % 5 == 0
        puts "foobar"
      elsif x % 3 == 0 
        puts "foo"
      elsif x % 5 == 0
        puts "bar"
      else
        puts x
      end
  end
end

puts "Enter a number"
input = gets.chomp.to_i
puts "okay its foobar time!"
foobar(input)
